# Variasi Hook — A/B Test (Threads Rekapan Mitra)

Hook = baris pertama, penentu orang berhenti scroll atau lanjut. Tiap draft inti (lihat `00-strategi.md` §6) punya **4 variasi hook**. Badan post tetap sama; ganti pembukanya, posting di hari berbeda, lihat mana yang paling banyak engagement.

**Cara pakai A/B test:**
1. Pilih 1 draft, posting Hook A.
2. Minggu depan, post badan yang sama dengan Hook B.
3. Catat: like, reply, share, profile visit.
4. Hook pemenang → pakai pola serupa untuk draft lain.

## Pola hook yang menang (referensi)

Empat pola dipakai di atas — perhatikan polanya, bukan kalimatnya:

1. **Adegan spesifik** ("Jam 11 malam, anak udah tidur…") → paling relatable, scroll-stopper.
2. **Pertanyaan langsung** ("Untung bersihmu bulan ini berapa?") → ngundang refleksi + reply.
3. **Pernyataan kontras** ("Laku ≠ untung", "Bedanya bukan effort. Bedanya sistem.") → bikin penasaran.
4. **Kutipan/teaser** ('"…tinggal forward chat doang."') → janji cerita, narik baca lanjut.

Saat bikin draft baru, coba keempat pola ini dulu sebelum nulis badan post.
